## Lists 
Types of lists
- Ordered list ```<ol>```
- Unordered lists ```<ul>```
- List Items ```<li>```
- Definition Lists ```<dl>```
    - Term ```<dt>```
    - Definition ```<dd>```
- Nested lists
    - Put second list into a ```<li>``` element
